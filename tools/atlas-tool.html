Hermes Atlas & Avatar Tool

Single-file tool for inspecting the hermes_dark Live2D model's texture atlas and generating an avatar PNG cropped from the live render. Open this in the same browser you use for the VTuber (localhost:12393) so the model files are already served. No server of our own required — it's all client-side using the Cubism SDK + pixi-live2d-display.
